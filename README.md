This is an Ecommerce web application developed as part of my Advanced Web Programming course, built using React.js, Express, Node.js, and MongoDB.

The application allows users to:

- Browse and view product details

- Add products to cart (ATC)

- Checkout and place orders

Admin users can:

- Create, update, and delete products

- Manage products through admin authentication
